# RULES — Dead Tactics

Читай этот файл перед каждой задачей. Для деталей — смотри соответствующий документ.

---

## Абсолютные запреты

- **Не хардкодить** строки `'survivor'`, `'zombie'` — только `UNIT_TYPES.SURVIVOR`, `UNIT_TYPES.ZOMBIE`
- **Не хардкодить** урон, параметры юнитов, количество кадров — всё берётся из конфига
- **Не вызывать `render()`** внутри цикла движения — только `syncUnitsWithDOM()`
- **Не трогать `battle.js`** без явной задачи связанной с анимацией или рендером
- **Не создавать `id`** которые уже есть в DOM
- **Не менять `u.effects` напрямую** — только через `addEffect` / `removeEffect`
- **Не менять стейт боя** нигде кроме `state.js`
- **Не менять стейт кампании** нигде кроме `data.js`
- **Не дублировать логику** между файлами — каждая функция живёт в одном месте

## Чеклист перед любой задачей

1. Прочитать раздел документации который относится к задаче
2. Написать план изменений — какие файлы трогаешь и почему
3. Получить подтверждение плана перед Act Mode
4. После изменений: проверить в браузере, потом коммит

---

## Чеклист после изменений

- [ ] Нет ошибок в консоли браузера
- [ ] Функциональность работает визуально
- [ ] `git diff` — убедиться что изменения применились
- [ ] Коммит только после успешного теста
